":yuricry:" (2.34) - if you do not raid with a BRD

This set is for if you aren’t raiding with a bard or you don’t know what you’ll be raiding with.

March 7th (2.33) - if you do raid with a BRD

The DH buff from BRD makes this set better than the other, it’s suggested to use the non-BRD set if you’re doing pugs.

Rampardos Theorem

This set is for people who are more comfortable with slower sets.
